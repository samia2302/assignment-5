Question-1: What are the different ways to select an element in the DOM?

Ans: getElementById(),getElementsByClassName(),getElementsByTagName(),querySelector(),querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent?

Ans: innerHTML - gets/sets the html content including the tags inside an element, innerText - gets/sets only the visible text inside an element, textContent - gets/sets all the text including hidden ones of an element but doesn't include html tags.

Question-3: What is event delegation in the DOM?

Ans: It is a technique where it adds a single event listener to a parent element instead of adding event listener to multiple child elements.

Question-4: What is event bubbling in the DOM?

Ans: It means if you click on a child element,the event will first be handled by the child, then by its parent and so on,up to the root of the document.

Question-5: How do you create,add and remove elements using JavaScript?

Ans: To create - createElement(), To add - appendChild(), To remove - removeChild().
